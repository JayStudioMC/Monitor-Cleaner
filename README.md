> **あなたのデスクトップが、きれいに。**

Monitor Cleanerは、マルチモニター環境での作業をより快適に、よりスタイリッシュにするための自動ウィンドウ管理ツールです。

---

## ✨ 主な機能

- **🖼 スタイリッシュな自動最小化**
  モニターからマウスが離れると、そのモニターにあるウィンドウを自動的に最小化。常にクリーンなデスクトップを保ちます。
- **🛡 便利な除外機能**
  作業中の資料や動画再生ウィンドウなど、最小化されたくないアプリを簡単に除外設定可能。
- **🎥 スマート操作検知 (Pro)**
  PCから離れた際の全モニター一括クリーンアップや、動画再生中の自動保留など、より高度な自動化を提供します。

[Releases](https://github.com/your-username/monitor-cleaner/releases) ページから最新の `MonitorCleaner_Setup.exe` をダウンロードしてください。

## 🛠 開発者向け

このプロジェクトは Python と PyQt6 で構築されています。

```bash
# クローン
git clone https://github.com/your-username/monitor-cleaner.git

# 依存関係のインストール
pip install -r requirements.txt

# 実行
python main.py
```
