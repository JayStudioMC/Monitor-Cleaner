Features

スタイリッシュなデスクトップへ

モニターからマウスが離れると、一定時間後にそのモニターにあるウィンドウを自動最小化。雑多なデスクトップから卒業できます。好きな壁紙を設定したけどウィンドウが邪魔で見えない方にもピッタリ。

便利な除外機能

資料や動画再生、フルスクリーンのウィンドウなど自動的に最小化したら困るものは設定から除外できます。不要なウィンドウのみを最小化してスマートなデスクトップに

スマートな操作検知

PCを離れたら全モニターをクリーンアップ。急な離席でもデスクトップのプライバシーをスマートに守ります。

ライセンスプラン

基本機能は永久無料。高度なカスタマイズはPro版で。
