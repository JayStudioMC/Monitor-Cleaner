　Monitor Cleaner - スマート・モニター・クリーナー
　Version 1.0.0

【アプリ概要】
Monitor Cleanerは、PC作業をより快適にするための自動ウィンドウ管理ツールです。
「モニターからカーソルが離れた時」や「PCを一定時間操作していない時」に、
そのモニターにあるウィンドウを自動的に最小化し、デスクトップをクリーンに保ちます。

マルチモニター環境で、使っていないモニターのウィンドウが気散じになるのを防いだり、
離席時にデスクトップを隠したりするのに最適です。
プライバシーの保護や背景画像を見たい時にぜひ使ってください！

【主な機能】
・モニター別・自動最小化：カーソルが別のモニターに移動し一定時間が経つと、元のモニター上にあるウィンドウを最小化
・スマート動画検知：YouTube等の動画再生中は最小化を自動的に保留
・全体非アクティブ検知：マウス・キーボード操作が止まった際に全ウィンドウを最小化
・除外ウィンドウ設定：特定のアプリ（音楽プレイヤーやチャット等）を最小化対象から除外

【無料版とPro版の違い】
本ソフトは基本無料でお使いいただけますが、Pro版（有料）へアップグレードすることで

【ライセンス認証方法】
1. タスクトレイの「Monitor Cleaner」アイコンを右クリックし「設定」を選択します。
2. 「ライセンス」タブを開きます。
3. 購入したライセンスキーを入力し「保存」を押すとPro機能が有効化されます。

【動作環境】
・OS: Windows 10 / 11 (64bit)
・Python 3.10以降のランタイム（ビルド済み版は不要）
